# 📘 Любители русского языка — мини-приложение для MAX

Мини-приложение для платформы MAX, посвящённое русскому языку и грамотности.  
Проект реализован как SPA (Single Page Application) и предназначен для запуска внутри чат-бота MAX.

На текущем этапе реализован каркас приложения с навигацией по разделам без наполнения контентом.

---

## 🧩 Разделы приложения

1. ⚠️ **НА ГРАБЛЯХ** — типичные ошибки и ловушки русского языка  
2. 🔍 **ГРАМОТЕЙКА** — правила, примеры, разборы  
3. 🎮 **СЛОВОЗОР** — игровые задания и викторины  
4. 🗣 **БОЛЬШОЙ СЛОВАРЬ** — словарь слов и выражений  

(Контент будет добавлен на следующих этапах разработки)
